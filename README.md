# Virtual Patient Simulator (CUBA)

## Overview  
The Virtual Patient Simulator (CUBA) is an AI-powered system designed to help medical students practice clinical communication, diagnostic reasoning, and patient interaction in a controlled environment.  

The system simulates real-world doctor–patient conversations using Natural Language Processing (NLP) and Machine Learning models, enabling realistic multi-turn interactions.

---

## Objectives  
- Improve doctor-patient communication (ATCOM skills)  
- Enhance diagnostic reasoning ability  
- Provide a safe environment for clinical practice  
- Enable feedback-driven learning  

---

## Features  
- AI-based virtual patient interaction  
- Voice input and output support  
- Context-aware multi-turn conversations  
- Disease prediction and validation module  
- Feedback and explanation system  
- Multilingual interaction support  

---


## Tech Stack  

Frontend  
- HTML, CSS, JavaScript  
- Eel  

Backend  
- Python  

AI and Machine Learning  
- LLM: Phi-3 Mini, ClinicalBERT  
- Speech Processing: speechrecognition 
- Text-to-Speech: pyttsx3 


---

## Usage  

1. Start the application  
2. Interact with the virtual patient using text or voice  
3. Ask clinical questions  
4. Analyze responses  
5. Predict the disease  
6. Receive feedback and explanation
